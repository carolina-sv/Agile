# **App Name**: Agile

## Core Features:

- Project Creation: Allow users to create new projects with defined goals, timelines, and team members.
- Task Management: Enable users to create, assign, and track tasks with statuses, deadlines, and priorities.
- Kanban Board: Visualize project workflow using a Kanban board with drag-and-drop functionality to move tasks between statuses.
- Team Collaboration: Facilitate team communication through task-specific comments, mentions, and file sharing.
- Reporting: Generate reports and track project progress, team performance, and potential bottlenecks.
- AI Task Assigner: Suggest the optimal person to assign to a task, based on individual skills and current workload; uses a tool to consult past assignments.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), promoting focus and productivity.
- Background color: Very light cyan (#E0FFFF), creating a clean and calm workspace.
- Accent color: Lime green (#32CD32), for task completion confirmations and highlights.
- Font: 'Inter', a sans-serif font, for a modern, neutral, and readable interface.
- Use a set of minimalist icons representing various tasks and project categories.
- Prioritize a clean, intuitive layout with clearly defined sections for projects, tasks, and team communications.
- Incorporate subtle transitions and animations to enhance user experience during task updates and workflow changes.
